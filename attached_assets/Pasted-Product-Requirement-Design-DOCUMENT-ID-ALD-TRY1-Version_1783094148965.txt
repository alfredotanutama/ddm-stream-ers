DDM STREAMER
 
1 OVERVIEW
Product Name: COBOL Stream Tool
Problem Statement: COBOL developers saat ini harus membuka COBOL editor, mengedit field via copybook, export ke .txt, baru bisa hit/test stream. Proses ini repetitive dan time-consuming.
Goal: Web app yang membantu COBOL developer untuk:
-          Generate stream dari copybook
-          Decompose/view stream berdasarkan copybook
Tech Stack:
-          Next.js + Tailwind CSS
-          Pure frontend, no backend, no database, no login
 
2 USER FLOW
Fitur 1 - Stream Generator:
User buka tab "Generator"
User upload atau paste copybook (.txt)
App parse copybook → tampilkan list field + input box per field
User isi value tiap field
App generate stream real-time
User copy atau download stream sebagai .txt
Fitur 2 - Stream Decomposer:
User buka tab "Decomposer"
User upload atau paste copybook (.txt)
User upload atau paste stream
App decompose stream → tampilkan field + value 1:1
User bisa lihat hasil decomposition
 

 
3 FEATURES DETAIL
Tab 1: GENERATE
Upload copybook via file upload (.txt) atau paste text area.
Parse copybook dan tampilkan field table:
o   Field name
o   PIC type
o   Length
o   Input box untuk isi value
Generate stream otomatis/real-time saat user mengetik
Output stream ditampilkan di text area
Tombol Copy to Clipboard
Tombol Download as .txt
 
Tab 2: DECOMPOSE
Upload copybook via file upload (.txt) atau paste text area.
Upload stream via file upload (.txt) atau paste text area
Tampilkan hasil decomposition dalam bentuk table:
o   Field name
o   PIC type
o   Length
o   Value (hasil pecahan dari stream)
REDEFINES ditampilkan sebagai sub-group dengan indentasi
Tombol Copy result (copy table as text)
